# Day 10 - Working with CSV Files

## What is a CSV File?

A CSV (Comma-Separated Values) file is a simple file used to store data in rows and columns.

Each row represents a record, and each column represents a field.

## Why Use CSV Files?

* Easy to read and write
* Stores structured data
* Supported by most data tools
* Commonly used in Machine Learning projects

## Common Operations

### Read CSV File

Load data into a program for analysis.

### View Data

Check the first few rows and understand the dataset.

### Select Columns

Choose only the required columns.

### Save Data

Store the updated data in a new CSV file.

## Applications

* Student Records
* Sales Data
* Customer Information
* Financial Data
* Healthcare Data

## Workflow

CSV File → Load Data → Analyze Data → Process Data → Save Data

## Important Points

* CSV files store data in tabular format.
* Each row represents one record.
* Each column represents one feature.
* CSV is one of the most used file formats in Data Science.
* Always check the data before analysis.

## One-Line Summary

A CSV file stores data in rows and columns and is widely used for data analysis and Machine Learning projects.
