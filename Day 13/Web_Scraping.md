# Day 13 - Web Scraping

## What is Web Scraping?

Web Scraping is the process of collecting data from websites automatically.

It helps us gather large amounts of data for analysis and Machine Learning projects.

## Why Use Web Scraping?

* Collect website data
* Save time and effort
* Build datasets
* Analyze online information

## Common Uses

* Product Price Tracking
* News Collection
* Job Listings
* Real Estate Data
* Market Research

## Workflow

Website → Send Request → Get Web Page → Extract Data → Save Data → Analyze Data

## Advantages

* Collects large amounts of data
* Faster than manual data collection
* Helps create custom datasets
* Useful for Data Science and Machine Learning

## Important Points

* Web Scraping extracts data from websites.
* Always check a website's terms of use before scraping.
* Clean the collected data before analysis.
* Scraped data can be stored in CSV, JSON, or a database.
* Web Scraping is widely used in data collection projects.

## One-Line Summary

Web Scraping is the process of automatically collecting data from websites for analysis and Machine Learning.
