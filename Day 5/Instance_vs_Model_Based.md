# Day 5 - Instance Based vs Model Based Learning

## Instance Based Learning

Instance Based Learning stores training data and makes predictions by comparing new data with stored examples.

It learns from similar instances.

### Examples

* K-Nearest Neighbors (KNN)
* Recommendation Systems

## Model Based Learning

Model Based Learning creates a model from training data and uses that model for predictions.

It learns patterns from the data.

### Examples

* Linear Regression
* Logistic Regression
* Decision Trees

## Difference

| Instance Based                 | Model Based           |
| ------------------------------ | --------------------- |
| Stores training data           | Builds a model        |
| Uses similarity for prediction | Uses learned patterns |
| Slower prediction              | Faster prediction     |
| Requires more memory           | Requires less memory  |

## Real Example

Instance Based Learning = Solving a new problem by looking at similar old problems.

Model Based Learning = Learning a rule and using it to solve new problems.

## Important Points

* Instance Based Learning remembers data.
* Model Based Learning learns patterns.
* KNN is an Instance Based algorithm.
* Regression algorithms are Model Based.
* Model Based Learning is commonly used in Machine Learning.

## One-Line Summary

Instance Based Learning predicts using stored examples, while Model Based Learning predicts using a learned model.
