# Day 9 - Tensors

## What is a Tensor?

A Tensor is a data structure used to store and process data in Machine Learning and Deep Learning.

Tensors are similar to arrays but can have multiple dimensions.

## Types of Tensors

### Scalar (0D Tensor)

A single value.

Example:

* Age = 20

### Vector (1D Tensor)

A list of values.

Example:

* [10, 20, 30]

### Matrix (2D Tensor)

Rows and columns of values.

Example:

|   |   |   |
| - | - | - |
| 1 | 2 | 3 |
| 4 | 5 | 6 |

### Higher-Dimensional Tensor

A tensor with 3 or more dimensions.

Example:

* Images
* Videos
* Deep Learning data

## Why Tensors are Important?

* Store data efficiently
* Perform fast calculations
* Used in Deep Learning models
* Supported by frameworks like TensorFlow and PyTorch

## Real Example

A grayscale image can be stored as a 2D tensor.

A color image can be stored as a 3D tensor.

A video can be stored as a 4D tensor.

## Important Points

* Tensor is the basic data structure in Deep Learning.
* Scalars, vectors, and matrices are all tensors.
* Tensors can have multiple dimensions.
* Deep Learning frameworks use tensors for computations.
* Most ML and DL operations are performed on tensors.

## One-Line Summary

A Tensor is a multi-dimensional data structure used to store and process data in Machine Learning and Deep Learning.
