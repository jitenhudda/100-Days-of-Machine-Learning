# Day 14 - Understanding Your Data

## What is Data Understanding?

Data Understanding is the process of exploring and examining a dataset before building a Machine Learning model.

It helps us understand the structure and quality of the data.

## Why is it Important?

* Understand the dataset
* Find missing values
* Detect incorrect data
* Identify patterns and trends

## What to Check?

### Dataset Shape

Know the number of rows and columns.

### Features

Understand the meaning of each column.

### Data Types

Check whether the data is numeric, categorical, or text.

### Missing Values

Find columns with empty or null values.

### Duplicate Data

Check for repeated records.

## Workflow

Load Data → Explore Data → Check Data Quality → Understand Features → Prepare for Analysis

## Important Points

* Always understand the data before training a model.
* Good data leads to better predictions.
* Check data types and missing values.
* Remove duplicate or incorrect data.
* Data understanding is the first step of data analysis.

## One-Line Summary

Understanding your data helps you explore, clean, and prepare the dataset before building a Machine Learning model.
