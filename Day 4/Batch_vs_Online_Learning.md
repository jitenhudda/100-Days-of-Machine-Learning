# Day 4 - Batch Learning vs Online Learning

## Batch Learning

Batch Learning trains a model using the complete dataset at one time.

The model does not learn automatically after deployment.

### Examples

* House Price Prediction
* Sales Forecasting
* Exam Result Prediction

## Online Learning

Online Learning trains a model continuously using new data.

The model keeps improving as new data arrives.

### Examples

* Recommendation Systems
* Fraud Detection
* Stock Market Analysis

## Difference

| Batch Learning        | Online Learning            |
| --------------------- | -------------------------- |
| Uses all data at once | Uses new data continuously |
| Trained once          | Updated regularly          |
| Slower updates        | Faster updates             |
| Best for fixed data   | Best for changing data     |

## Important Points

* Batch Learning uses the complete dataset.
* Online Learning learns from new data.
* Batch Learning needs retraining for updates.
* Online Learning updates automatically.
* Online Learning is useful for real-time systems.

## One-Line Summary

Batch Learning learns from all data at once, while Online Learning learns continuously from new data.
