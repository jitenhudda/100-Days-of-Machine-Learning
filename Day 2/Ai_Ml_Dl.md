# Day 2 - AI vs ML vs DL

## Artificial Intelligence (AI)

AI means making machines smart enough to perform tasks that normally require human intelligence.

Examples:
- Chatbots
- Self-driving cars
- Virtual assistants

## Machine Learning (ML)

ML is a subset of AI.

It allows machines to learn from data and improve automatically.

Example:
- Movie recommendation system
- Spam email detection

## Deep Learning (DL)

DL is a subset of ML.

It uses neural networks with many layers to learn complex patterns.

Examples:
- Face Recognition
- Voice Assistants
- Image Classification

## Relationship

AI
└── ML
    └── DL

## Difference Table

| AI | ML | DL |
|----|----|----|
| Makes machines intelligent | Learns from data | Uses deep neural networks |
| Broad field | Subset of AI | Subset of ML |
| Can use rules | Uses data | Uses huge data |
| Simple to complex tasks | Prediction tasks | Complex tasks |

## Real Example

AI = Entire Human Brain

ML = Learning Process

DL = Deep Learning Power of Brain

## Important Points

- AI is the biggest field.
- ML is inside AI.
- DL is inside ML.
- Every DL is ML.
- Every ML is AI.
- But every AI is not ML.

## One-Line Summary

AI makes machines smart, ML helps them learn from data, and DL learns using deep neural networks.
