# Day 12 - Fetching Data from API

## What is an API?

An API (Application Programming Interface) allows different applications to share data with each other.

It helps us get real-time data from websites or online services.

## Why Use APIs?

* Get real-time data
* Access online services
* Automate data collection
* Use data in ML projects

## Common Uses

* Weather Data
* Movie Information
* Stock Market Data
* Sports Scores
* News Updates

## Workflow

API Request → Receive Data → Process Data → Analyze Data → Build ML Model

## Advantages

* Provides real-time data
* Saves time and effort
* Easy to access online data
* Supports automation

## Important Points

* API stands for Application Programming Interface.
* APIs connect different applications.
* Most APIs return data in JSON format.
* APIs are widely used in Data Science and Machine Learning.
* An internet connection is usually required to fetch API data.

## One-Line Summary

An API helps applications share data and provides real-time information for analysis and Machine Learning projects.
