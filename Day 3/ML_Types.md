# Day 3 - Types of Machine Learning

## Definition

Machine Learning is mainly divided into three types based on how data is provided.

## 1. Supervised Learning

The model learns using labeled data.

Labeled means the answer is already available.

### Example

Input: Student Marks

Output: Pass/Fail

The model learns from previous examples.

### Applications

- House Price Prediction
- Spam Detection
- Exam Result Prediction

---

## 2. Unsupervised Learning

The model learns from unlabeled data.

No correct answer is given.

The model finds hidden patterns on its own.

### Example

Customer Grouping

### Applications

- Customer Segmentation
- Market Analysis
- Pattern Detection

---

## 3. Reinforcement Learning

The model learns by trial and error.

It receives rewards for correct actions and penalties for wrong actions.

### Example

Training a Robot

### Applications

- Self-driving Cars
- Game Playing AI
- Robotics

---

## Simple Comparison

| Type | Data | Answer Available |
|--------|--------|--------|
| Supervised | Labeled | Yes |
| Unsupervised | Unlabeled | No |
| Reinforcement | Reward Based | No |

## Easy Example

Supervised:
Teacher gives questions and answers.

Unsupervised:
Teacher gives only questions.

Reinforcement:
Teacher gives rewards for correct answers.

## Important Points

- Supervised Learning → Labeled Data
- Unsupervised Learning → Unlabeled Data
- Reinforcement Learning → Reward and Penalty
- Supervised Learning is the most commonly used type.

## One-Line Summary

Machine Learning has three main types: Supervised, Unsupervised, and Reinforcement Learning.
