# Day 15 - Univariate Analysis

## What is Univariate Analysis?

Univariate Analysis is the process of analyzing **one variable** at a time.

It helps us understand the distribution and characteristics of a single feature.

## Why is it Important?

* Understand individual features
* Find data distribution
* Detect outliers
* Identify missing values

## Common Techniques

### Numerical Data

* Mean
* Median
* Mode
* Minimum and Maximum

### Categorical Data

* Count of each category
* Frequency of values

## Common Charts

* Histogram
* Bar Chart
* Pie Chart
* Box Plot

## Applications

* Data Exploration
* Data Cleaning
* Feature Understanding
* Exploratory Data Analysis (EDA)

## Workflow

Load Data → Select One Feature → Analyze Values → Visualize Data → Draw Insights

## Important Points

* Univariate Analysis studies only one variable.
* It is the first step of Exploratory Data Analysis (EDA).
* It helps identify patterns and outliers.
* It improves data understanding before model building.
* Both numerical and categorical data can be analyzed.

## One-Line Summary

Univariate Analysis is the process of analyzing a single variable to understand its distribution, patterns, and data quality.
