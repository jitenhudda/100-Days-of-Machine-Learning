# Day 11 - Working with JSON and SQL Files

## What are JSON and SQL Files?

JSON and SQL are common formats used to store and manage data in Data Science and Machine Learning.

## JSON File

JSON (JavaScript Object Notation) stores data in a structured format using key-value pairs.

### Uses

* Store application data
* Exchange data between systems
* Work with APIs

## SQL Database

SQL (Structured Query Language) is used to store, manage, and retrieve data from databases.

### Uses

* Store large amounts of data
* Retrieve specific records
* Manage business data

## Difference

| JSON                              | SQL                                    |
| --------------------------------- | -------------------------------------- |
| Stores data in key-value format   | Stores data in tables                  |
| Best for structured data exchange | Best for managing large databases      |
| Easy to read                      | Fast data retrieval                    |
| Commonly used in APIs             | Commonly used in business applications |

## Applications

* Web Applications
* E-commerce Systems
* Banking Systems
* Healthcare Records
* Machine Learning Projects

## Workflow

JSON / SQL Database → Load Data → Analyze Data → Process Data → Build ML Model

## Important Points

* JSON stores data in key-value pairs.
* SQL stores data in tables.
* JSON is widely used in APIs.
* SQL is used to manage databases.
* Both formats are important for Data Science and Machine Learning.

## One-Line Summary

JSON is used to store structured data, while SQL is used to manage and retrieve data from databases.
